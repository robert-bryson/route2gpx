🗺️ Route to GPX

Convert routes to GPX files for GPS devices

🔒 Privacy First

This app runs entirely in your browser. There is no server — your API key and routes are stored only in your browser's localStorage and are never transmitted anywhere except directly to Google's API.

⌨️ Keyboard Shortcuts

  • Enter in any field — Get route
  • Ctrl+Enter — Get route from anywhere
  • Escape — Exit click-to-add mode
  • Arrow keys — Pan map when focused
  • + / - — Zoom map

💡 Tips

  • Click the ✚ button on the map to add points by clicking
  • Enter coordinates as lat, lng (e.g., 40.7128, -74.0060)
  • Routes persist when you refresh the page
  • GPX files work with Garmin, Wahoo, Strava, and most GPS apps
